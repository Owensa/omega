How (most) Cryptocurrency Mining Works

The production of cryptocurrencies isn't anything like that of regular money

Request for Transaction

Someone requests to send cryptocurrency through the network. They input the amount they want to send along with a transaction fee they wish to pay.

The Request is broadcast to all miners in the network

The transaction is sent and bundled as part of a block (which include many other transactions as well) and broadcasted to the network to be verified or 'mined'.

Miners jointly validate the transaction

They take the 'blocks' or lumpsum of transactions, verify them as legitimate through mathematical operations using an algorithm and spread them in the network.

Confirmations

Once the transaction has been "mined" or verified, it is broadcast on the network or 'blockchain' and considered completed. Transaction are 'reconfirmed' to ensure validity.

Miners are given newly minted cryptocurrency as a reward for this work (until there are no more coins left to mint otherwise known as a hard cap, at that point miners are given the necessary transaction fees to collect).

Transaction complete. Block is added permanently to the blockchain.

After a few confirmations, the transaction is considered irreversible and complete. At this point the block is added to the network or 'blockchain' and this process goes on and on.
